# Calculadora de Aposentadoria em Python

Este projeto foi desenvolvido em Python com o objetivo de calcular:

- Idade do usuário
- Tempo de contribuição
- Idade mínima para aposentadoria
- Elegibilidade para aposentadoria

## Funcionalidades

- Validação de dados de entrada
- Cálculo automático da idade
- Verificação de tempo de contribuição
- Regra para trabalho de risco
- Estruturas condicionais e loops
- Tratamento de erros com try/except

## Tecnologias utilizadas

- Python
- datetime

## Objetivo

Projeto desenvolvido para prática de lógica de programação e construção de portfólio na área de tecnologia.
